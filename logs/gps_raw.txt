    Timestamp - ISO 8601 UTC time
    STATUS    - VALID or NO FIX
    SAT       - Satellites tracked
    NMEA      - 1Hz pulse count (seconds since start)
    ms        - Always 0 in 1 PPS mode (output at boundary)

TIMING: Use message arrival as 1 PPS reference
─────────────────────────────────────────────────────────────────

COMMANDS: Send 's' for status

PYTHON SAMPLE:
─────────────────────────────────────────────────────────────────
  import serial, time
  s = serial.Serial('COM6', 115200)
  while True:
      line = s.readline().decode().strip()
      rx_time = time.time()  # 1 PPS reference
      if 'VALID' in line:
          print(f'{rx_time:.3f}: {line}')
